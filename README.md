# React Add Marks
1. **Clone the repository**
```bash
git clone https://github.com/your-username/student-management-system.git
cd student-management-system
install node dependencies
```
```bash
npm install
```
```bash
npm start
```
A simple React-based **Student Management System** that allows users to view and add students with basic information such as name, age, and marks.

## 🚀 Features

- ✅ View list of students
- ➕ Add new students
- ⚛️ Uses React's `useState` for state management
- 🧩 Modular component structure (`AddStudentForm`, `StudentList`)

## 🧱 Technologies Used

- React
- JavaScript
- HTML/CSS (basic styling)
